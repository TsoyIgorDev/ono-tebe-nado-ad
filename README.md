## Проект «Оно тебе надо» — Аукцион вещей

[![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)](https://developer.mozilla.org/ru/docs/Web/HTML)
[![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)](https://developer.mozilla.org/ru/docs/Web/CSS)
[![Figma](https://img.shields.io/badge/Figma-F24E1E?style=for-the-badge&logo=figma&logoColor=white)](https://figma.com)

Веб-страница для аукциона необычных вещей, созданная в рамках учебного проекта. Проект демонстрирует современные подходы к вёрстке с использованием семантической разметки, CSS Grid и Flexbox.

## Особенности

- **Семантическая HTML5 разметка** – правильное использование тегов для улучшения доступности
- **CSS Grid Layout** – сложные сетки для шапки, футера и секций
- **Flexbox** – гибкое выравнивание элементов
- **Pixel Perfect** – точное соответствие макету
- **Адаптивная типографика** – кастомные шрифты Raleway
- **Переиспользуемые компоненты** – оверлей, карточки, кнопки
- **Оптимизированные изображения** – форматы WebP и оптимизированные JPG
